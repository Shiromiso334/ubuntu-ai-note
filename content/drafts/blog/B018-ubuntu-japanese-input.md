Ubuntuをインストールした直後、「日本語が入力できない」と困ったことはないでしょうか。
自分も最初にこの壁にぶつかりました。
Windowsでは当たり前にできていたことが、Ubuntuではひと手間必要な場合があります。

この記事では、Ubuntuで日本語入力を使えるようにするための確認手順をまとめています。

---

## この記事の対象読者

**対象読者**

- Ubuntuをインストールしたばかりの方
- 日本語入力の切り替え方が分からない方
- 文字入力まわりで最初につまずきたくない方

**この記事でわかること**

- Ubuntuで日本語入力が使えるしくみ
- 入力切り替えの基本的な操作
- うまく打てないときに確認するポイント

---

Ubuntuでは、インストール時に「日本語」を選んでいれば、日本語入力エンジンの「Mozc（モズク）」が自動で入っていることが多いです。
ただし、設定が有効になっていない場合や、入力ソースが追加されていない場合は使えないことがあります。
手順を確認しながら進めていきましょう。

## Ubuntuの日本語入力で最初に確認したいこと

### 日本語入力は最初から使えるのか

Ubuntuで日本語入力を使うには、「日本語入力エンジンが入っていること」と「入力ソースとして設定されていること」の2点が必要です。

インストール時に日本語を選んでいれば、Mozc（Googleが開発した日本語入力エンジン）が標準で入っていることがほとんどです。
ただし「最小インストール」を選んだ場合や、インストール後に設定を変えた場合は入っていないことがあります。

### 入力切り替えの基本

日本語入力と英語入力の切り替えは、キーボードの「半角/全角」キー、または `Super（Windowsキー）+ Space` で切り替えられます。

Ubuntuのデフォルト設定では `Super + Space` が入力ソースの切り替えに割り当てられています。
日本語キーボードを使っている場合は「半角/全角」キーが使えることが多いです。

---

## 日本語入力を使う手順

### 1. キーボード設定を確認する

まず、キーボードのレイアウトが「日本語」になっているか確認します。

1. 画面右上のシステムメニューを開く
2. 「設定」を選ぶ
3. 「キーボード」を開く
4. 「入力ソース」に「日本語（Mozc）」が追加されているか確認する

「日本語（Mozc）」がなければ、`+` ボタンから追加してください。
検索欄に「日本語」と入力すると表示されます。

### 2. Mozc の有無を確認する

「日本語（Mozc）」が選択肢に出てこない場合、Mozcがインストールされていない可能性があります。
ターミナルを開いて以下のコマンドで確認・インストールできます。

```bash
sudo apt install ibus-mozc
```

インストール後、一度ログアウトして再度ログインすると設定に反映されます。

### 3. 入力ソースを切り替える

「日本語（Mozc）」が設定に追加できたら、実際に切り替えて試してみましょう。

1. テキストエディタやブラウザのアドレスバーなど、文字を入力できる場所をクリックする
2. `Super + Space` または「半角/全角」キーを押す
3. 画面右上または入力欄の近くに「あ」と表示されれば、日本語入力モードになっています

「A」が表示されているときは英数入力モード、「あ」が表示されているときは日本語入力モードです。

## うまく入力できないときの確認ポイント

### 英数モードに戻れないとき

日本語入力中に英語を打ちたいとき、そのまま入力すると確定前の状態で止まることがあります。
そういうときは `Escape` キーを押して入力を取り消し、「半角/全角」や `Super + Space` で英数モードに切り替えてください。

Mozcのアイコンを使う方法もあります。
画面右上や入力欄付近に表示されているMozcのアイコンを右クリックすると、入力モードの選択肢が出てきます。「直接入力（英数）」を選べば、キーボードショートカットなしで切り替えられます。

### 日本語候補が出ない

ひらがなは入力できるのに変換候補が出ない場合は、以下を確認してみてください。

- Mozcが「ひらがな」モードになっているか（「直接入力」になっていると変換されません）
- スペースキーを押して変換を試みているか（Mozcはスペースキーで変換候補を表示します）

設定を変えた直後にうまく反映されていない場合は、一度ログアウトして再ログインしてみましょう。これで解決するケースが多いです。

---

## まとめ

Ubuntuで日本語入力を使うために確認するのは、主に2点です。

1. 「日本語（Mozc）」が入力ソースとして設定されているか
2. 入力の切り替えが `Super + Space` または「半角/全角」キーでできるか

インストール時に日本語を選んでいれば、追加のインストールなしで使えることが多いです。
うまくいかない場合は `ibus-mozc` のインストールから試してみましょう。
