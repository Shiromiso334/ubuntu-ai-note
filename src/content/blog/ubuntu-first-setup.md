Ubuntuのインストールが完了すると、ログイン画面が表示されます。
「インストールできた、でも次に何をすればいい？」と迷う方は多いです。
自分も最初は、どこから手を付ければいいかわからずしばらく画面を眺めていました。

この記事では、インストール直後に確認しておきたい項目を順番にまとめています。

インストール直後のUbuntuは、ほぼ使える状態になっています。
ただ、いくつか確認しておかないと後で気になる点が出てきます。
最初に見ておくべきことを、一つずつ確認していきましょう。

## Ubuntuを入れた直後に最初に確認したいこと

### インターネット接続

まず、インターネットに接続できているか確認します。
画面右上に接続状況を示すアイコンが表示されています。

有線LANを使っている場合は自動で繋がっていることが多いです。
Wi-Fiを使う場合は、右上のアイコンをクリックして接続先を選んでください。

接続できているかどうかは、ブラウザ（Firefoxが最初から入っています）を開いて試してみると分かります。

### ソフトウェア更新

インターネットに繋がったら、最初にソフトウェアの更新を確認します。
インストール直後のUbuntuは、リリース後のアップデートが当たっていないことがあるためです。

更新の確認は「ソフトウェアとアップデート」アプリから行えます。
または、ターミナルを開いて以下のコマンドを実行する方法もあります。

```bash
sudo apt update && sudo apt upgrade -y
```

`sudo`（スードゥ）は管理者権限でコマンドを実行する命令です。
実行するとパスワードを求められます。インストール時に設定したパスワードを入力してください。

更新が完了したら、再起動してみましょう。

### 日本語表示・時刻の確認

インストール時に日本語を選んでいれば、システムは日本語で表示されているはずです。
念のため以下を確認してください。

- 画面右上の時刻が正しいか
- システムのメニューやアプリ名が日本語表示になっているか

日本語表示が乱れている場合は「設定」→「地域と言語」から確認できます。

## 最初に見直したい設定

### 電源設定

Ubuntuはデフォルトで、一定時間操作しないと画面が消えたりスリープに入ります。
作業中に画面が暗くなってしまう場合は、電源設定を調整しましょう。

1. 右上のシステムメニュー（電源ボタンのようなアイコン）を開く
2. 「設定」を選ぶ
3. 「電源」を開く
4. 「画面オフにするまでの時間」や「自動サスペンド」を好みに合わせて変更する

---

### ソフトウェアとアップデート

先ほどの `apt update` とは別に、「ソフトウェアとアップデート」の設定画面で、更新の通知方法を確認しておくと安心です。

「セキュリティアップデートを自動的にダウンロードしてインストールする」を有効にしておくと、重要な更新を見逃しにくくなります。

---

### Dock と外観

Ubuntuのデスクトップ左端には「Dock」と呼ばれるアプリランチャーが表示されています。
最初から使いやすいですが、好みに合わせて位置やサイズを変えることもできます。

設定の場所：「設定」→「外観」→「Dock」

位置を画面下や右にしたり、アイコンの大きさを変えることができます。
ここは使いながら少しずつ調整していく場所なので、最初は触らなくても問題ありません。

## 後回しでよいこと

### 細かい見た目の調整

テーマ変更・壁紙・フォントのカスタマイズなどは、後からいつでもできます。
最初は標準の設定のまま使い始め、慣れてきたら変えていきましょう。

### 高度な開発環境の構築

プログラミング環境やAIツールの導入は、基本操作に慣れてから進めた方がスムーズです。
まずはブラウザやファイルマネージャーの使い方を確認するところから始めましょう。

## まとめ

インストール直後にやるべきことは、大きく3つです。

1. インターネット接続の確認
2. ソフトウェア更新
3. 電源設定の確認

この3点を済ませておけば、ひとまず安心して使い始められます。
日本語入力の設定など、実際に文字を打つ場面で必要なことは次の記事で確認しましょう。
